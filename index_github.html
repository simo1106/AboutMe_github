實務專案經驗

1. 基於 ESP32-S3 與 Android 之邊緣運算導盲避障系統 —「你是我的眼」
TinyML 邊緣運算 / FreeRTOS 實時多工 / 多軌網路通訊架構整合
  • 邊緣影像辨識： 使用 Edge Impulse 平台訓練輕量化 FOMO 物件偵測模型，以純 C++ 函式庫原生部署於 ESP32-S3 端，在完全斷網環境下，仍可自主即時辨識紅綠燈與斑馬線。
  • 高併發韌體架構： 在 ESP-IDF 環境下導入 FreeRTOS 任務架構。運用 Mutex（互斥鎖）防止週邊資源衝突，並透過 Queue（隊列）在多個任務間安全傳遞 ICM20948 IMU 的高速六軸加速度與陀螺儀結構體數據。
  • 多軌通訊架構整合： 運用 HTTP 註冊專屬 URI 結構體建立內部 WebServer 進行 OV2640/OV3660 即時路況影像串流；獨立架設 TCP Socket 確保超聲波避障之緊急警報具備高即時與低延遲性；透過 JSON 打包設備指標並以 MQTT 上傳雲端以便進行遠端穩定性追蹤。
  • 行動端 API 與資料庫整合： 於 Android Studio 開發專屬導航 App。設計爬蟲擷取 OpenStreetMap 交叉路口資訊並存於本地 SQLite 資料庫；整合 Google Routes API 計算導航路線轉折點，以語音引導視障人士前置避障。
2. 微控制器藍牙與物聯網自動化警報系統
盛群單晶片開發 / 藍牙通訊介面映射 / 雲端自動化連動
  • 微控制器端韌體開發： 使用 C 語言對盛群 HT66F2390 微控制器進行全系統底層配置，精準定義系統時脈、手動完成 UART 腳位映射與鮑率參數設定。
  • 訊號處理與調變： 擷取環境感測器之類比訊號並進行 ADC 數位轉換，將轉換結果透過 HC-05 藍牙模組進行高穩定度之無線串列傳輸。
  • 雲端連動自動化警報： 於 MIT App Inventor 2 平台建構行動端接收中樞，設計條件判斷邏輯（如當 ADC > 3800 時）。全自動驅動後端 SmtpClient 發送防護電子郵件，並連動 Line Bot 送出即時簡訊，落實物聯網全自動化防護監控。
3. 獨立自主專案：自製智慧空氣清淨機
軟硬體端到端開發 / 電路板焊接 / 流體密封結構優化 / 成本控管
  • 電路與控制實作： 自學 SPWM 弦波脈寬調變理論基礎，獨立設計馬達驅動電路，並手工完成高可靠度之電路板焊接；結合微處理器控制電路，達成手動/粉塵濃度自動雙模控制與無級調速。
  • 流體與結構優化： 負責 3D 外殼建模設計。針對清淨機核心的「短循環」痛點，特別設計了由木條與塑膠膜組成的風管密封結構，並黏貼海綿填充物以優化風道，確保過濾效能，同時預留未來易拆卸清洗之結構。

4. 畢業實務專案：智慧化室內空氣品質控制模組建構
感測器訊號處理 / 自動化控窗結構
  • 針對室內二氧化碳與懸浮微粒 (PM2.5) 濃度危害，利用微控電路整合感測器訊號處理。當室內活動人員過多導致 CO2 濃度超標時，微處理器程式會自動運算並驅動硬體機械結構，調節窗戶開度大小以引入外氣換氣，實現智慧化室內空氣品質管理。
